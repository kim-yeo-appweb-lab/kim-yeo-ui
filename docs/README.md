### Foundation (Storybook)

Storybook에서 확인 가능한 디자인 시스템 기초 문서:

- **Introduction** - 프로젝트 소개 및 빠른 시작
- **Colors** - 색상 팔레트 및 토큰 시스템
- **Typography** - 폰트 패밀리, 크기, 무게, 행간
- **Spacing** - 간격 시스템 (Padding, Margin, Gap)
- **Accessibility** - 접근성 가이드 및 WAI-ARIA 표준
